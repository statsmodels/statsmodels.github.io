statsmodels.regression.mixed_linear_model.MixedLM.loglike¶

MixedLM.loglike(params)[source]¶

Evaluate the (profile) log-likelihood of the linear mixed effects model.

Parameters:

params : MixedLMParams, or array-like.

The parameter value. If array-like, must be a packed parameter vector compatible with this model.
Returns:

The log-likelihood value at params.

Notes

This is the profile likelihood in which the scale parameter scale has been profiled out.

The input parameter state, if provided as a MixedLMParams object, can be with respect to any parameterization.
