    class statsmodels.tsa.statespace.varmax.VARMAXResults(model, params, filter_results, cov_type=None, cov_kwds=None, **kwargs)[source]¶

    Class to hold results from fitting an VARMAX model.

    Parameters:¶
    modelVARMAX instance

    The fitted model instance

    Attributes:¶
    specificationdictionary

    Dictionary including all attributes from the VARMAX model instance.

    coefficient_matrices_varndarray

    Array containing autoregressive lag polynomial coefficient matrices, ordered from lowest degree to highest.

    coefficient_matrices_vmandarray

    Array containing moving average lag polynomial coefficients, ordered from lowest degree to highest.
