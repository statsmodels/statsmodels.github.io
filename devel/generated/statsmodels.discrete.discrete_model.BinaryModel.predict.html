statsmodels.discrete.discrete_model.BinaryModel.predict¶

BinaryModel.predict(params, exog=None, linear=False)[source]¶

Predict response variable of a model given exogenous variables.

Parameters

  • params (array-like) – Fitted parameters of the model.

  • exog (array-like) – 1d or 2d array of exogenous values. If not supplied, the whole exog attribute of the model is used.

  • linear (bool, optional) – If True, returns the linear predictor dot(exog,params). Else, returns the value of the cdf at the linear predictor.

Returns

Fitted values at exog.

Return type

array
