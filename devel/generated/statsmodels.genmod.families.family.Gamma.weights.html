statsmodels.genmod.families.family.Gamma.weights¶

Gamma.weights(mu)¶

Weights for IRLS steps

Parameters

mu (array-like) – The transformed mean response variable in the exponential family

Returns

w – The weights for the IRLS steps

Return type

array

Notes

\[w = 1 / (g'(\mu)^2 * Var(\mu))\]
