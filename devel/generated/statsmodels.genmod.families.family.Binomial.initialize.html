statsmodels.genmod.families.family.Binomial.initialize¶

Binomial.initialize(endog, freq_weights)[source]¶

Initialize the response variable.

Parameters

  • endog (array) – Endogenous response variable

  • freq_weights (array) – 1d array of frequency weights

Returns

  • If endog is binary, returns endog

  • If endog is a 2d array, then the input is assumed to be in the format

  • (successes, failures) and

  • successes/(success + failures) is returned. And n is set to

  • successes + failures.
